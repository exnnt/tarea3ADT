Campos Obligatorios

  • Dirección: Dirección completa de entrega.
  • Localidad: Ciudad o localidad de entrega.
  • Peso: Peso del paquete (en kg). Debe ser un número positivo (Ej: 2.5).
  • Dimensiones: Largo, ancho y alto del paquete (en cm). Deben ser números enteros positivos.

Opciones Adicionales

  • Envío Urgente: Marca la casilla si el envío es urgente.

Confirmar

  • Crear Envío: Haz clic en "Confirmar" para registrar el envío.

Si todos los campos son válidos, se confirmará el envío y se redirigirá a la pantalla principal.
